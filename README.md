# 📖 About The Project

This is a Full Stack **E-Commerce Web Application** developed using **Flask** and **Firebase Firestore**.

The application provides a complete online shopping experience including user authentication, product management, shopping cart functionality, order placement, and order history. The project is deployed on **Render** for public access.
